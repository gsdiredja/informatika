1. Alur Struktur Kode Proyek yang Rapi

Sebuah aplikasi proyek lintas bidang (misalnya aplikasi penghitung nilai rata-rata pelajaran atau simulasi sains) umumnya dibagi menjadi 3 blok utama:

Bagian Blok Tugas Kode Analogi Nyata
1. Input Data Menerima angka atau teks yang dimasukkan oleh pengguna lewat keyboard. Memasukkan bahan-bahan mentah ke dalam wadah masak.
2. Proses (Logika) Mengolah data menggunakan rumus matematika, percabangan (if), atau perulangan. Kompor menyala memasak bahan mentah sesuai langkah resep.
3. Output Tampilan Menampilkan hasil olahan data ke layar monitor agar bisa dibaca pengguna. Makanan matang disajikan cantik di atas piring saji.

2. Lembar Kerja Praktik Siswa (Google Colab)

Mari kita coba membuat satu prototipe proyek lintas bidang sederhana: **Aplikasi Kasir Kantin Sekolah Digital**. Salin kode terstruktur ini ke Google Colab kelompokmu, jalankan, dan perhatikan bagaimana tiap blok bekerja saling mendukung! 

# ====================================================
# BLOK 1: FUNGSI PROSES (LOGIKA REKOMENDASI HARGANYA)
# ====================================================
def hitung_total_belanja(harga_satuan, jumlah_beli):
    total = harga_satuan * jumlah_beli
    
    # Memberikan diskon jika membeli lebih dari 5 porsi (Percabangan)
    if jumlah_beli > 5:
        diskon = total * 0.10  # Diskon 10%
        total = total - diskon
        print("🎉 Selamat! Kelompokmu dapat diskon tim 10%")
        
    return total

# ====================================================
# BLOK 2: INPUT DATA (MENERIMA MASUKAN PENGGUNA)
# ====================================================
print("--- SISTEM KASIR KANTIN DIGITAL KELOMPOK ---")
nama_makanan = input("Masukkan nama menu makanan: ")
harga_menu   = int(input("Masukkan harga satu porsi (Rp): "))
jumlah_porsi = int(input("Mau beli berapa porsi?: "))

# ====================================================
# BLOK 3: EKSEKUSI PROSES DAN TAMPILKAN OUTPUT
# ====================================================
total_bayar = hitung_total_belanja(harga_menu, jumlah_porsi)

print("\n================ STRUK BELANJA ================")
print(f"Menu Pesanan : {nama_makanan}")
print(f"Banyaknya    : {jumlah_porsi} porsi")
print(f"Total Bayar  : Rp {total_bayar}")
print("===============================================")
