Selamat! Kamu sudah sampai di materi terakhir pada modul ini. Sekarang, bayangkan kamu memiliki resep rahasia membuat es krim cokelat. Daripada menulis ulang seluruh langkah pembuatan setiap kali ada pelanggan yang memesan, kamu cukup memberikan nama resep itu: "BikinEsKrim()". Di dunia pemrograman, resep rahasia ini dinamakan Fungsi (Function).

Konsep Utama Fungsi: Fungsi adalah sebuah blok kode terpisah yang sengaja dibuat untuk menyelesaikan tugas spesifik. Kode di dalam fungsi tidak akan berjalan sebelum kita memanggil namanya. Ini membuat kode kita menjadi rapi, ringkas, dan tidak ditulis berulang-ulang (*Don't Repeat Yourself* / DRY).

1. Cara Membuat dan Memanggil Fungsi di Python

Untuk membuat fungsi di Python, kita menggunakan kata kunci def (singkatan dari *define*), diikuti nama fungsinya, tanda kurung (), dan titik dua. 

# 1. Cara membuat fungsi
def sapa_siswa(nama):
    print(f"Halo {nama}, selamat belajar Informatika!")

# 2. Cara memanggil fungsi agar berjalan
sapa_siswa("Budi")
sapa_siswa("Siti")

2. Mengenal Array / Vector (Menggunakan List di Python)

Jika variabel biasa hanya bisa menyimpan **satu data** (misal: nilai = 80), maka Array atau di Python disebut List dapat menyimpan **banyak data sekaligus** di dalam satu kotak variabel. Ibaratnya seperti kotak pensil yang bisa diisi pulpen, penghapus, penggaris, dan pensil warna.

Ciri khas List/Array adalah menggunakan tanda kurung siku [ ] dan setiap datanya dipisahkan dengan tanda koma.

Operasi Array (List) Contoh Kode Python Penjelasan
Membuat List game = ["Roblox", "ML", "FF"] Membuat daftar nama-nama game.
Mengakses Data (Index) print(game[0]) Mengambil data pertama. Ingat! Komputer menghitung urutan dari angka 0.
Menambah Data game.append("Minecraft") Fungsi .append() memasukkan data baru ke barisan paling akhir.

3. Lembar Kerja Praktik Siswa (Google Colab)

Mari kita satukan konsep **Fungsi** dan **Array (List)** dalam satu program keren! Salin kode di bawah ini ke Google Colab-mu untuk menghitung rata-rata nilai rapor kelas secara otomatis. 

# 1. MEMBUAT FUNGSI HITUNG RATA-RATA
def hitung_rata_rata(daftar_nilai):
    total = sum(daftar_nilai)       # sum() menjumlahkan semua angka di list
    jumlah_data = len(daftar_nilai) # len() menghitung total item di list
    hasil = total / jumlah_data
    return hasil                    # Mengembalikan nilai hasil hitungan keluar fungsi

# 2. MEMBUAT ARRAY/LIST DATA NILAI SISWA
nilai_informatika = [85, 90, 78, 92, 88]

# 3. MEMANGGIL FUNGSI
rata_rapor = hitung_rata_rata(nilai_informatika)

print("============ KELOLA DATA NILAI ============")
print(f"Daftar Nilai Siswa : {nilai_informatika}")
print(f"Banyaknya Siswa    : {len(nilai_informatika)} anak")
print(f"Rata-rata Kelas    : {rata_rapor}")
print("===========================================")
