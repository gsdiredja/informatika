PEMROGRAMAN BAHASA C++

1. Pengertian Bahasa C++

Bahasa C++ adalah salah satu bahasa pemrograman paling populer yang bersifat compiled, berkinerja tinggi, dan mendukung paradigma Pemrograman Berorientasi Objek (Object-Oriented Programming / OOP).

C++ dikembangkan oleh Bjarne Stroustrup pada tahun 1979 di AT&T Bell Labs sebagai ekstensi dari bahasa C. Nama "C++" berasal dari operator increment ++ pada bahasa C, yang melambangkan bahwa C++ adalah versi peningkatan atau langkah berikutnya dari bahasa C.

Karakteristik Utama C++:

  • General-Purpose: Bisa digunakan untuk membuat berbagai jenis aplikasi (game engine, sistem operasi, aplikasi desktop, embedded system).
  • Multi-Paradigm: Mendukung pemrograman prosedural, berorientasi objek (OOP), dan generik.
  • Tingkat Menengah (Mid-Level Language): Memiliki efisiensi dan akses kontrol memori tingkat rendah (low-level) seperti bahasa C, namun memiliki abstraksi tingkat tinggi (high-level) seperti OOP.

2. Istilah-Istilah Penting pada C++

Berikut adalah istilah-istilah dasar hingga menengah yang wajib dipahami saat belajar C++:

A. Struktur & Komponen Dasar

Istilah Pengertian & Fungsi
Header File (#include) File pustaka (library) yang diimpor di awal program. Contoh: #include <iostream> untuk fungsi input-output.
std:: (Standard Namespace) Ruang nama tempat pustaka standar C++ berada, untuk mencegah bentrok nama variabel/fungsi.
main() Function Fungsi utama dan titik awal (entry point) di mana eksekusi program C++ selalu dimulai.
Compiler Program pembantu untuk menerjemahkan kode C++ (source code) menjadi file eksekusi (binary/machine code).
cout & cin cout (character output) untuk menampilkan data ke layar; cin (character input) untuk menerima input dari pengguna.

B. Variabel, Data, & Memori

Istilah Pengertian & Fungsi
Data Types Tipe data untuk menentukan jenis nilai variabel (misal: int, float, double, char, bool, string).
Pointer (*) Variabel khusus yang menyimpan alamat memori dari variabel lain.
Reference (&) Nama alias atau shortcut langsung ke alamat memori variabel yang sudah ada.
Array Kumpulan data dengan tipe data yang sama yang disimpan secara berurutan di memori.
Vector (std::vector) Struktur data mirip Array, tetapi ukurannya dinamis (bisa bertambah/berkurang secara fleksibel).

C. Pemrograman Berorientasi Objek (OOP)

C++ sangat terkenal dengan fitur OOP-nya. Berikut istilah pentingnya:

  • Class: Cetak biru (blueprint) atau templat untuk membuat sebuah objek.
  • Object: Bentuk nyata (instance) yang dibuat berdasarkan rancangan Class.
  • Encapsulation: Mengemas data dan fungsi menjadi satu unit (class) serta membatasi akses langsung dari luar menggunakan Access Specifiers (private, public, protected).
  • Inheritance (Pewarisan): Kemampuan sebuah class baru (child) untuk mewarisi sifat dan fungsi dari class lain (parent).
  • Polymorphism: Kemampuan suatu fungsi/objek untuk memiliki banyak bentuk atau perilaku berbeda.
  • Constructor: Fungsi khusus dalam class yang dipanggil secara otomatis saat objek pertama kali dibuat.
  • Destructor: Fungsi khusus yang dipanggil saat objek dihancurkan untuk membersihkan memori.

3. Komponen Input/Output dan Struktur Program

Komponen dasar dari bahasa C/C++ yang digunakan untuk komunikasi antara program dengan pengguna (proses Input/Output) serta pengaturan struktur program:

Perintah & File Header Utama:

  • #include: Perintah pra-pemrosesan (preprocessor directive) yang berfungsi untuk memasukkan/mengimpor library (pustaka kode) eksternal ke dalam program kita. Tanpa perintah ini, program tidak akan mengenali fungsi-fungsi dasar seperti printf atau scanf.
  • stdio.h: Kepanjangan dari Standard Input Output Header (.h berarti file header). Menyediakan fungsi-fungsi dasar input-output seperti printf() (menampilkan ke layar) dan scanf() (menerima input dari keyboard).
    Catatan: Biasanya ditulis #include <stdio.h> pada bahasa C, sedangkan pada C++ modern sering ditulis #include <iostream>.
  • int main(): Fungsi utama yang menjadi titik awal eksekusi program (entry point). Ketika program dijalankan, komputer akan selalu mencari dan mengeksekusi baris kode yang ada di dalam int main() terlebih dahulu dari atas ke bawah. Kata int di depannya menandakan fungsi ini akan mengembalikan nilai berupa angka bulat (biasanya return 0; di akhir program sebagai tanda program sukses berjalan).

Penentu Format / Format Specifier (%)

Simbol % yang diikuti huruf tertentu digunakan pada fungsi input-output (seperti printf atau scanf) untuk memberi tahu komputer jenis/tipe data apa yang sedang dicetak atau dibaca:

Format Specifier Tipe Data Pengertian & Kegunaan Contoh Nilai
%d atau %i int (Integer) Menampilkan/membaca bilangan bulat (desimal/signed integer). 10, -5, 100
%f float / double Menampilkan/membaca bilangan pecahan/desimal. 3.14, 0.5, -12.45
%c char (Character) Menampilkan/membaca satu karakter tunggal. 'A', 'z', '7', '#'
%s string / char[] Menampilkan/membaca kumpulan karakter / teks / kata. "Halo", "Koding"

4. Simbol-Simbol Penting dalam Koding C/C++

A. Karakter Lepas / Escape Sequence (Format Teks)

  • \n (New Line): Membuat baris baru (sama seperti menekan Enter di keyboard).
  • \t (Tab): Memberikan jarak/spasi tab horizontal.
  • \\: Menampilkan karakter backslash \.
  • \": Menampilkan tanda petik ganda " di dalam teks.

B. Simbol Sintaksis & Pemformatan

  • ; (Semicolon / Titik Koma): Penanda akhir pernyataan (statement). Setiap baris instruksi wajib diakhiri dengan ; agar tidak terjadi syntax error.
  • { } (Curly Braces / Kurung Kurawal): Menandai awal dan akhir dari blok kode atau fungsi.
  • & (Ampersand / Operator Alamat): Digunakan saat mengambil input dengan scanf("&variabel") untuk menandai alamat memori variabel tempat data tersimpan.
  • // dan /* ... */ (Komentar): Digunakan untuk menulis catatan di kode yang diabaikan oleh komputer (tidak dieksekusi). Gunakan // untuk komentar 1 baris, dan /* ... */ untuk komentar banyak baris.

5. Percabangan Logika (Conditional Statements / Control Flow)

Percabangan Logika adalah konsep dalam pemrograman yang memungkinkan komputer untuk mengambil keputusan berdasarkan kondisi tertentu.
Prinsip dasar: "Jika hari hujan, maka saya membawa payung. Jika tidak, saya tidak membawa payung."

Jenis-Jenis Percabangan Logika:

1. Percabangan if (1 Kondisi)

Digunakan ketika kita hanya ingin mengeksekusi suatu perintah jika kondisinya bernilai benar (True). Jika salah (False), program tidak akan melakukan apa-apa.
Logika: Jika [Kondisi True] → Jalankan Perintah. 

nilai = 85
if nilai >= 75:
    print("Selamat, Anda Lulus!")
2. Percabangan if - else (2 Kondisi)

Digunakan untuk menangani dua pilihan: satu aksi jika kondisi bernilai True, dan aksi alternatif jika kondisi bernilai False.
Logika: Jika [Kondisi True] → Jalankan Perintah A. Jika tidak → Jalankan Perintah B. 

int nilai = 60;
if (nilai >= 75) {
    cout << "Selamat, Anda Lulus!";
} else {
    cout << "Maaf, Anda Harus Remidi.";
}
3. Percabangan if - else if - else / elif (Banyak Kondisi)

Digunakan jika ada lebih dari dua pilihan kondisi yang perlu diperiksa secara berurutan.
Logika: Cek Kondisi 1 → Jika True, jalankan Perintah 1. Jika False, Cek Kondisi 2 → Jika True, jalankan Perintah 2. Jika semua False → Jalankan Perintah Terakhir (Else). 

int nilai = 85;
if (nilai >= 90) {
    cout << "Nilai Anda: A";
} else if (nilai >= 80) {
    cout << "Nilai Anda: B";
} else if (nilai >= 70) {
    cout << "Nilai Anda: C";
} else {
    cout << "Nilai Anda: D";
}
4. Percabangan switch - case

Metode alternatif dari if - else bercabang yang khusus digunakan untuk memeriksa satu variabel terhadap banyak nilai konstan/pasti. 

int kodePilihan = 2;
switch (kodePilihan) {
    case 1:
        cout << "Anda memilih Nasi Goreng";
        break;
    case 2:
        cout << "Anda memilih Mie Ayam";
        break;
    default:
        cout << "Pilihan tidak tersedia";
}

Operator Logika Pendukung Percabangan

Sering kali kita perlu mengecek beberapa syarat sekaligus menggunakan Operator Logika:

Operator Nama Operator Fungsi / Syarat Lulus Contoh
&& (atau and) AND Bernilai True jika SEMUA kondisi terpenuhi (umur >= 17 && punyaSIM == true)
|| (atau or) OR Bernilai True jika SALAH SATU kondisi terpenuhi (hari == "Sabtu" || hari == "Minggu")
! (atau not) NOT Membalikkan nilai logika (True jadi False, dan sebaliknya) !(nilai < 60)
