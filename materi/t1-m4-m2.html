1. Bagaimana Komputer Menggambar Lintasan Burung?

Di Materi 1, kita sudah memegang rumus fisikanya. Sekarang, bagaimana cara membuat komputer menggambar gerakan burung tersebut? Caranya adalah dengan menggunakan Algoritma Perulangan (Looping).

Bayangkan seperti kita membuat animasi kartun. Kita menyuruh komputer menghitung posisi burung di detik 0.0, lalu maju sedikit ke detik 0.2, detik 0.4, dan seterusnya menggunakan perintah while sampai burungnya menyentuh tanah kembali.

2. Konsep Deteksi Target (Collision Detection)

Tantangan seru lainnya adalah menentukan apakah burung kita berhasil mengenai target (babi) atau tidak. Komputer akan mengecek koordinat pendaratan terakhir burung dengan posisi target menggunakan logika if-else.

Batas Toleransi Jarak: Angka desimal di komputer sangat detail dan jarang menghasilkan angka yang sama persis 100% (misalnya burung mendarat di 39.98 meter, padahal target berada di 40.00 meter). Oleh karena itu, kita memberikan ruang toleransi (misal sebesar 2 meter). Jika selisihnya dekat, target dianggap tetap hancur!

3. Kode Program (Salin dan Jalankan di Google Colab)

Buka Google Colab kalian, lalu salin kode Python di bawah ini untuk melihat simulasinya secara langsung dalam bentuk grafik: 

import math
import numpy as np
import matplotlib.pyplot as plt

def visualisasi_dan_cek_target(v0, sudut_derajat, x_target, toleransi=2.0, g=9.8):
    # 1. Memasukkan rumus dari Materi 1
    sudut_rad = math.radians(sudut_derajat)
    v0_x = v0 * math.cos(sudut_rad)
    v0_y = v0 * math.sin(sudut_rad)
    t_total = (2 * v0_y) / g
    
    # 2. Membuat deretan waktu dari 0 sampai burung jatuh (100 titik data)
    t_array = np.linspace(0, t_total, 100)
    
    # 3. Menghitung semua koordinat lintasan burung
    x_lintasan = v0_x * t_array
    y_lintasan = (v0_y * t_array) - (0.5 * g * (t_array**2))
    
    # 4. Ambil jarak pendaratan paling ujung
    jarak_akhir = x_lintasan[-1]
    selisih = abs(jarak_akhir - x_target) # Mencari jarak absolut burung ke target
    
    # 5. Membuat grafik visual lintasan
    plt.figure(figsize=(10, 5))
    plt.plot(x_lintasan, y_lintasan, label='Lintasan Burung', color='blue')
    plt.scatter(x_target, 0, color='red', s=200, label='Target (Babi)', zorder=5)
    
    plt.title('Grafik Simulasi Pemrograman Lintas Bidang: Gerak Burung')
    plt.xlabel('Jarak Horizontal (meter)')
    plt.ylabel('Ketinggian Vertikal (meter)')
    plt.axhline(0, color='black', linewidth=1)
    plt.grid(True)
    
    # 6. Logika Kondisional (If-Else) untuk status kemenangan
    if selisih <= toleransi:
        print(f"🎯 TARGET TERKENA! Burung mendarat di {jarak_akhir:.2f}m.")
        plt.text(jarak_akhir/3, max(y_lintasan)/2, "💥 HIT!", fontsize=24, color='orange', weight='bold')
    else:
        print(f"❌ MELESET! Burung mendarat di {jarak_akhir:.2f}m. Target berada di {x_target}m.")
        plt.text(jarak_akhir/3, max(y_lintasan)/2, "⚪ MISS!", fontsize=24, color='purple', weight='bold')
        
    plt.legend()
    plt.show()

# COBA JALANKAN: Kecepatan awal 20 m/s, sudut 45 derajat, target di jarak 38 meter
visualisasi_dan_cek_target(v0=20, sudut_derajat=45, x_target=38)
