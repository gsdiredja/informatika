1. Operator Relasional & Logika (Kunci Pintu Gerbang)

Sebelum membuat percabangan, kita harus tahu cara membuat "pernyataan kondisi". Pernyataan ini diukur menggunakan operator matematika khusus yang menghasilkan nilai True (Benar) atau False (Salah).

Operator Arti Contoh di Python
== Sama dengan x == 5
!= Tidak sama dengan x != 10
> / < Lebih besar / Lebih kecil nilai > 75
>= / <= Lebih besar atau sama dengan / Lebih kecil atau sama dengan usia >= 17

2. Tiga Jenis Percabangan di Python

Di dalam bahasa Python, struktur percabangan ditulis menggunakan kata kunci if, else, dan elif. Yuk, kita bedah satu per satu!

A. Kondisi Tunggal (if)

Digunakan jika jalurnya hanya satu. Jika kondisi bernilai benar, kode dijalankan. Jika salah, program mengabaikannya dan lanjut ke bawah. 

if total_belanja > 100000:
    print("Selamat! Kamu mendapatkan diskon 10%")

B. Dua Kondisi Alternatif (if-else)

Digunakan ketika ada pilihan cadangan. Jika kondisi if tidak terpenuhi, maka blok else pasti akan dieksekusi. 

nilai = 78
if nilai >= 75:
    print("Status: LULUS")
else:
    print("Status: REMEDIAL")

C. Banyak Kondisi Pilihan (if-elif-else)

Digunakan jika pilihan opsinya lebih dari dua (bertingkat). Komputer akan mengecek dari atas, dan berhenti pada opsi pertama yang bernilai benar. 

usia = 15
if usia >= 17:
    print("Kategori: Dewasa (Bisa bikin KTP)")
elif usia >= 13:
    print("Kategori: Remaja")
else:
    print("Kategori: Anak-anak")
⚠️ Aturan Emas Python (Indentasi): Berbeda dengan bahasa lain yang menggunakan tanda kurung kurawal {}, Python mendeteksi cakupan kode percabangan menggunakan Indentasi (geseran spasi/tab ke dalam). Pastikan kode di bawah baris if menjorok ke dalam sebesar 4 spasi agar tidak memicu IndentationError!
