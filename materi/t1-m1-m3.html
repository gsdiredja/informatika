IDENTIFIKASI JENIS SEGITIGA

1. Menganalisis dan Memahami Masalah (Problem Analysis)

Masalah Guru matematika memberi tugas untuk mengidentifikasi sebuah segitiga.
Deskripsi Diberikan 3 buah bilangan bulat pada rentang 1 – 1000 yang merupakan panjang sisi sebuah segitiga, untuk menentukan apakah ketiga bilangan tersebut (a, b, c) membentuk sebuah segitiga beserta jenisnya.
Masukan (Input) 3 buah bilangan bulat a, b, c pada rentang 1 – 1000.
Proses & Keluaran Memeriksa apakah data merupakan sisi segitiga dengan keluaran hasil:
  • Segitiga sama sisi
  • Segitiga sama kaki
  • Segitiga sembarang
  • Bukan Segitiga

2. Merancang Strategi / Algoritma (Algorithm Design)

Untuk semua segitiga dengan panjang sisi a, b, dan c harus memenuhi ketaksamaan segitiga:

  • a + b > c
  • a + c > b
  • b + c > a

3. Mengimplementasikan Algoritma ke Kode (Implementation)

Perkakas program Bahasa C++: 

/* Program untuk Menentukan Jenis Segitiga */
#include <stdio.h>

int main()
{
    int a, b, c;
    scanf("%d %d %d", &a, &b, &c);
    
    if ((a >= b + c) || (b >= a + c) || (c >= a + b))
        printf("Bukan Segitiga");
    else
        if ((a == b) && (b == c))
            printf("Segitiga Sama Sisi");
        else if ((a == b) || (b == c) || (a == c))
            printf("Segitiga Sama Kaki");
        else
            printf("Segitiga Sembarang");
    return 0;
}

4. Menguji dan Membenahi Program (Testing & Debugging)

Setelah program selesai, kita perlu menguji kebenaran program tersebut menggunakan kasus uji berikut:

No Masukan Keluaran Sebenarnya Keluaran Program Hasil Benar/Salah
1 5 5 5 Segitiga Sama Sisi
2 1000 1000 500 Segitiga Sama Kaki
3 100 200 500 Bukan Segitiga
4 100 400 200 Segitiga Sembarang
